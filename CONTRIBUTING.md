# Contributing

Thanks for helping improve `jobsearch-skill`.

## Development

Use Node.js 20 or newer.

```bash
npm install
npm run release:check
```

## Pull Requests

- Keep changes focused and fixture-backed when behavior changes.
- Run `npm run release:check` before opening a PR.
- Avoid committing candidate-specific private data.

## Issues

Please include the command, sample input shape, and expected versus actual output.
